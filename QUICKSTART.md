## For Complete Beginners

### What You Need
1. **Roblox Studio** - Download from roblox.com/create
2. **Rojo** - A tool to sync code from VS Code to Roblox Studio
3. **VS Code** (optional but recommended) - Code editor

### Setup Steps

#### Option 1: Quick Build (Easy)
1. Open PowerShell in this folder
2. Run: `rojo build -o FeyGarden.rbxl`
3. Double-click the generated `FeyGarden.rbxl` file
4. Roblox Studio will open with your game
5. Press F5 to play!

#### Option 2: Live Sync (For Development)
1. Open PowerShell in this folder
2. Run: `rojo serve`
3. Keep PowerShell open
4. Open Roblox Studio
5. Install Rojo plugin from Roblox Library
6. Click "Connect" in the Rojo plugin
7. Your code will sync automatically when you save files!

### First Time Playing

When you start the game:
1. You spawn with **200 coins** and **starter seeds**
2. Your **personal garden plot** is created automatically
3. Click the **ðŸ›’ SHOP** button to buy more seeds
4. Click a **seed in your inventory** (bottom left) to select it
5. Click on the **brown floor of your garden** to plant
6. Wait for your plant to grow (watch the weather!)
7. Click **fully grown plants** to harvest and earn coins
8. Buy a **pet** from the shop for 300 coins to boost growth

### Tips
- **Rainy weather** makes plants grow faster!
- **Stormy weather** makes them grow even faster!
- Plants go through 3 stages: Seed â†’ Sprout â†’ Bloom
- Only bloomed plants can be harvested
- You can have up to 9 plants per plot

### Troubleshooting

**"rojo command not found"**
- Install Rojo first: https://rojo.space/docs/v7/getting-started/installation/

**"Can't connect to Rojo in Studio"**
- Make sure `rojo serve` is running in PowerShell
- Install the Rojo plugin in Roblox Studio
- Check if port 34872 is blocked by firewall

**"Nothing happens when I click"**
- Make sure you selected a seed from inventory first
- Click on the brown floor part of YOUR plot (has your name on sign)
- Check the Output window (View â†’ Output) for errors
